METHODOLOGY

Identification of Dog Breed and suggesting Foods and Vitamins section of the paper describes how the data sets are gathered and the implementation methodology used to recognize dog breeds and make predictions for the foods and vitamins that can be given to the pet dog, depending on their breed, age, and weight. To implement the dog breed identification part, a breed-classified data set was used. The data set contained around 10000 images in total. To implement the function, several steps were used. A data set is a compilation of data. A data set is, alternatively, the elements of a single database column or statistical data matrix, in which each column of the table represents a particular variable, and each row represents a particular component of the under-discussion data set. Machine learning initiatives need a training data set. The data set was what the model was trained on to carry out different tasks. Training is the stage of machine learning that is most important. You feed the special set to your model of machine learning during training to find patterns and make predictions. As a result, the model gains knowledge from the data and can do the given task. Over time, the model gets better at forecasting. By placing the model towards the test on brand new data, this is accomplished. The unseen data is taken from the testing set into which we previously divided our data. Our team won’t get an accurate answer if tested on the same data that was utilized for training because the model has become familiar with the results and has already seen the same patterns. You will receive a disproportionally high level of accuracy as a result. The framework for breed recognition was constructed using the aforementioned stages. The model carried out more than 92 of percentage for training accuracy, which was also proven when making test predictions on this function. To implement the food and vitamin suggestion part, real-time data, captured from a veterinarian was used. Once the user uploads a picture of their pet dog, the model makes a prediction image, and the age and the weight are taken as user input. Using all these parameters, the prediction will be made for the foods and vitamins. This function is fully integrated and is available in a single mobile application.
